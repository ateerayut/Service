## ⚡ SUPER QUICK COMMANDS

```powershell
# Create migration
dotnet ef migrations add MyMigrationName `
    --project src/Service.Infrastructure/Service.Infrastructure.csproj `
    --startup-project src/Service.Api/Service.Api.csproj

# Apply migration
dotnet ef database update `
    --project src/Service.Infrastructure/Service.Infrastructure.csproj `
    --startup-project src/Service.Api/Service.Api.csproj
```

## 🎯 The 3-Step Process

### Step 1️⃣: Modify Entity
Edit your domain entity (e.g., add a property)

### Step 2️⃣: Update DbContext
Configure the change in `AppDbContext.cs`

### Step 3️⃣: Create & Apply Migration
```powershell
# Create
dotnet ef migrations add YourMigrationName ...

# Apply
dotnet ef database update ...
```

**Done!** Your database is updated. ✅
